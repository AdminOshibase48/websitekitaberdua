# Website Pasangan Romantis

Website pribadi untuk pasangan yang berisi cerita, galeri foto, countdown anniversary, dan musik latar.

## Fitur Utama

1. **Countdown Anniversary** - Hitung mundur menuju anniversary berikutnya
2. **Pemutar Musik** - Musik latar yang dapat dikontrol
3. **Timeline Cerita** - Perjalanan hubungan pasangan
4. **Galeri Foto** - Koleksi foto kenangan
5. **Kenangan Spesial** - Momen-momen penting
6. **Desain Responsif** - Tampilan optimal di semua perangkat
7. **Animasi Halus** - Transisi dan efek visual yang menarik

## Warna yang Digunakan

- **#FFE4EF** - Pink muda (warna utama)
- **#B8DB80** - Hijau muda (warna sekunder)
- **#FF6B9D** - Pink cerah (warna aksen)
- **#333333** - Abu-abu gelap (warna teks)
- **#FFFFFF** - Putih (warna latar)

## Cara Menggunakan

1. Ganti data placeholder dengan data pasangan yang sebenarnya
2. Tambahkan foto-foto ke folder `assets/images/couple/`
3. Ganti file musik di folder `assets/music/`
4. Sesuaikan tanggal anniversary di file `countdown.js`
5. Ubah teks dan konten sesuai dengan cerita pasangan

Lihat bagian atas dokumen ini untuk struktur folder lengkap.

## Teknologi yang Digunakan

- HTML5
- CSS3 (dengan variabel CSS dan Flexbox/Grid)
